# Introdução

Este projeto é um teste técnico para a empresa Veloon, desenvolvendo uma API backend simples utilizando Node.js com Express.js.

### A API oferece duas funcionalidades principais:
- Listagem de Quartos Disponíveis: Exibe quartos disponíveis utilizando dados mockados, sem conexão com banco de dados.

- Criação de Reservas: Permite criar reservas armazenadas em memória, sem persistência entre sessões.

- O foco é demonstrar a capacidade de construir uma API funcional e eficiente com uma estrutura de código clara e organizada.


### Tecnologias Utilizadas

- Node.js
- Express.js
- TypeScript
- Vitest (para testes)


## Como Executar

- 1 Clone o projeto
```
git clone https://github.com/EmanuelVogt/veloon-hiring-test-backend.git
```
- 2 Instale as dependencias
```
npm install
```
- 3 Realize o build
```
npm run build
```
- 4 Rode localmente
```
npm run start
```

### o servidor será aberto na url http://localhost:3000

## Testes
